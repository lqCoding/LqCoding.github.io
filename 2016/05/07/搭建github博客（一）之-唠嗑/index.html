(原创) 搭建github博客(一)之初识github pages

github
  • github
  • 博客

为什么要写博客？写博客的三个阶段;github pages是什么鬼？

1.为什么要写博客？

1、写博客可以记录我们平时学习的知识，加深我们的对知识点的巩固，也为了以后再次碰到此类问题可以帮助自己快速查阅资料去解决问题。可以理解成我们的一个网络笔记。
2、其次本着开源分享的精神，我们可以把对一些技术独特的见解发表出来，为别人提供帮助，可以相互讨论。所以搭建自己的一个个人博客还是极好滴。

2.写博客的三个阶段

第一阶段:注册免费空间写博客
刚接触Blog，觉得很新鲜，试着选择一个免费空间来写，国内比较有名的像CSDN，博客园。注册一个账号，选择一个主题就可以写博客了，有点是简单省事，确点就是限制比较多，自己如果想实现更多的扩展功能基本很难实现。
第二阶段:搭建独立博客
发现免费空间限制太多，就自己购买的域名和空间想怎么折腾就怎么折腾，想要什么的样的布局和功能自己都可以写，优点就是完全自定义。确定就是太折腾了，维护起来太麻烦了。
第三阶段:别人帮你管理，你只负责写
觉得独立博客的管理太麻烦，最好在保留控制权的前提下，让别人来管，自己只负责写文章。这就是github博客，下面来说说github是个什么鬼….
但是很多人都停留在了第一阶段，因为第二和第三阶段，因为第二和第三阶段有一定的技术门槛，其次真的是挺折腾的。。。

####3. github是什么鬼？github pages是什么鬼？
github现在是全球最大的开源项目托管网站，从这个网站上你可以找到很多别人托管的开源项目，当然你也可以把你的开源项目托管到这个网站，
别人也可以访问你托管的项目。
github pages就是github给你托管的静态网站,咱们可以用来做博客，管理的事情github统统帮你做好了 ，还有免费空间，你需要做的就是做一个静态网站（你也可以让github帮你生成），发布到github即可。

4.搭建github博客需要准备的工作

1.注册github 账号 并创建仓库
2.安装npm和node.js
3.安装git
(考虑到准备工作会影响到篇幅，所以我将以上3步准备工作拆分到3篇博客中了，请看我的第二，三，四篇博客，请做好以上的准备工作之后继续往下看。)

5.简单利用git 获取github上的博客，修改并发布
