创建你的第一篇博客

hexo文件目录介绍：
scaffolds ：此文件夹内放的都是一些布局，markdown格式。
source_posts：此文件夹下放你的博客文件,为纯文本markdown 文件。
themes:这个文件夹下放的是主题这个不解释….
markdown语法学习
https://segmentfault.com/markdown
自己如何写博客以及hexo的运行原理：
hexo 可以将你写的纯文本的markdown文件编译成静态的HTML文件，并且支持markdown 语法，这就使得我们的可以使用Markdown语法去控制我们的博客内容。
我们可以使用文本编辑软件也可以使用markdown编辑器去写我们的博客。可以本地实时预览，然后再发布到网上。因为hexo的运行原理是去解析你的md文件，所以你的md文件必须符合规范 ，要不解析的时候会报错，也会导致编译不成功运行失败，运行失败会导致整个博客网站都无法发布，除非你删除有语法错误的博客，hexo会根据你的博客去生成标题，目录等等信息，我们可以看下一些在hexo里面常用的标签。
