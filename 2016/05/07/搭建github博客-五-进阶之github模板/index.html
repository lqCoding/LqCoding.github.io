搭建github博客进阶之github模板生成网站

经过“hello world”页面我们可以github博客的基本运行原理了，原理就是github免费提供域名空间免费给我们托管，我们只需要把我们搭建好的博客内容推送到github即可，但是每个页面都一点点的去写毕竟太麻烦了，其实github准备好了很多好看的静态模板，我们完全可以利用github提供的模板就可以生成一个既简单有漂亮的博客了，like this……
进入到github你的博客项目下 点击设置

点击自动模板向导

创建一个站点 默认值即可

选择你喜欢的模板 并发布站点

打开浏览器 查看效果 终于好看点了。。。

我们再次从github上把博客clone下来：

文件夹是这样子的，每次为了发表一篇博客，就得做一个静态网页，这也太麻烦了吧？？其实github也不建议你这么做，在刚才运行的模板下面是这个写的：

github推荐你用一个Jekyll的框架，去生成你的静态博客。jekyll是一个静态网站生成器，它可以将你写的纯文本文件直接渲染成静态网站！
是不是很神奇？跟jekll类似的静态网站生成器还有Hexo ，git+hexo也是现在用的比较多的博客解决方案，因为hexo 的模板比较多，还有支持的插件也比较多，
所以我选择的是hexo 框架，原理和使用方法跟jekll大同小异。下篇博客我将介绍hexo的使用方法。。。
